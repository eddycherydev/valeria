JWT Authentication

Valeria supports JSON Web Token (JWT) authentication to securely authenticate users via stateless tokens. JWTs are compact, URL-safe tokens containing claims that the server can verify and trust.

Generating JWTs

After a user logs in, generate a JWT with a secret key and user payload: 

<?php
use Valeria\Auth\JWT;

$payload = [
    'sub' => $user->id,
    'email' => $user->email,
    'iat' => time(),
    'exp' => time() + 3600 // 1 hour expiration
];

$token = JWT::encode($payload, getenv('JWT_SECRET'));
echo $token;

Verifying JWTs

To protect routes, use middleware that verifies the token's validity and decodes its payload: 

<?php
use Valeria\Auth\JWT;
use Valeria\Http\Request;

class AuthMiddleware
{
    public function handle(Request $request, $next)
    {
        $token = $request->header('Authorization');
        if (!$token || !JWT::verify($token, getenv('JWT_SECRET'))) {
            http_response_code(401);
            echo json_encode(['error' => 'Unauthorized']);
            exit;
        }

        return $next($request);
    }
}

JWT Middleware Usage

Apply the #[Middleware('auth')] attribute on controller methods to require JWT authentication: 

#[Middleware('auth')]
public function getProfile()
{
    // User is authenticated with a valid JWT
}

Configuration

Set your secret key in the .env file: 

JWT_SECRET=your-super-secret-key
Note: Always keep your JWT secret safe and never expose it publicly.
