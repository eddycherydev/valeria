Welcome to Valeria Framework

Valeria is a lightweight PHP framework built for API-first development and console tooling, with elegant routing, middleware support, JWT authentication, and a modern ORM called Lucid.

This documentation will guide you through the core features of Valeria, including how to define routes, apply middleware, protect your API with JWT tokens, and organize your project for maximum maintainability.

Whether you want to build scalable RESTful APIs or CLI tools, Valeria aims to give you a smooth, enjoyable developer experience inspired by the best modern PHP and JavaScript frameworks.
